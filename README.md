# Adya Vidyarthi — Portfolio Website

A multi-page personal portfolio site showcasing my background, skills, projects, and achievements — built with HTML, CSS, and JavaScript.

🔗 *Live Demo:* https://portfolio-ten-alpha-60.vercel.app/

## Pages
- *Home* — Landing page / intro
- *About* — Background and bio
- *Skills* — Technical skills overview
- *Projects* — Showcase of my work
- *Achievements* — Awards, certifications, milestones
- *Contact* — Resume, LinkedIn, GitHub, and contact details

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
